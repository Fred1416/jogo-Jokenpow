# ✊🖐✌️ Jogo Jokenpô (Pedra, Papel e Tesoura)

Um jogo clássico de Pedra, Papel e Tesoura desenvolvido totalmente do zero! Neste projeto, o usuário joga contra a "Alexa" (a máquina) e um placar automático registra quem está ganhando.

## 📸 Demonstração do Projeto

![Tela do Jogo](./img/Jogo%20Joken%20Pow.png)

## 🚀 Tecnologias Utilizadas

Esse projeto foi construído usando a base clássica do Front-end:
- **HTML5:** Estrutura e marcação da página e dos botões de jogar.
- **CSS3:** Estilização, layout e responsividade.
- **JavaScript:** Toda a lógica pesada por trás do jogo (sortear a jogada da máquina, comparar quem ganhou e atualizar o placar).

## 🧠 Aprendizados

Essa foi uma ótima oportunidade para praticar:
- Manipulação do DOM (alterar coisas na tela usando o Javascript).
- Criação de lógicas condicionais (`if`, `else`) para definir regras de vitória.
- Funções em JavaScript para modular o código (`playHuman`, comparar resultados, etc).

## 🛠️ Como rodar o projeto

1. Faça o clone (ou baixe) este repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
3. Clique nas mãos e tente ganhar da Alexa! 😂
